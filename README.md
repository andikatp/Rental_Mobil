## üéâ Features
### Customer App:
**User Authentication:**
- Login with email and password
- Register new accounts

**Product Management:**
- Browse car by category
- Search for car by city
- Search for car by availability in date
- View product details
- Book Car
  
**Security:**
- Secure authentication and data transmission (HTTPS)

### CMS App:
**Admin Authentication:** Only Admin can login cms<br>
**Rental Management:** CRUD product<br>
**Car Management:** CRUD brand<br>
